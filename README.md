# televize #
Script to run stream of Czech Television (Česká televize) in mplayer.

### Dependencies ###
 * python 2.7
 * curl
 * mplayer
 * bash

### Usage ###
```sh
./televize 1  # CT1
./televize 2  # CT2
./televize 4  # CT4
./televize 24  # CT24
```

Based on Perl scripts `televize` and `ctstream` from Petr Písař (http://xpisar.wz.cz).
